# LaunchGuard

LaunchGuard is a feature flag and progressive delivery platform built in Go.

It helps engineering teams safely release features using percentage-based rollouts, user targeting, kill switches, observability metrics, and automated rollback workflows.

## Tech Stack

- Go
- PostgreSQL
- Redis
- Kafka
- Docker
- Kubernetes
- Prometheus
- Grafana
- GitHub Actions

- Initial Go API setup
